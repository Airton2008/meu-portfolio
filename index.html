SandroCurioflix

O código usa três linguagens principais: HTML, CSS e um pouco de JavaScript (embutido nos elementos, como os vídeos do YouTube).

HTML é a estrutura do seu site, como se fosse o "esqueleto". Ele define: A organização do conteúdo, como títulos (h1), parágrafos (p), seções (section), entre outros. A incorporação de elementos externos, como vídeos do YouTube (iframe). Links e imagens, usados na seção de vídeos recomendados (a e img).

CSS cuida do design e da aparência do site, dando estilo e organização. No seu código, ele: Define cores e fontes, alterando o fundo (background), cor do texto (color) e fonte (font-family). Controla espaçamentos e posicionamento, usando margin, padding e display: flex; para alinhar os elementos corretamente. Cria efeitos interativos, como mudanças de opacidade e bordas ao passar o mouse nas imagens (hover). Melhora a experiência do usuário, com estilos mais atraentes e fáceis de visualizar

Embora não haja um código JS direto, ele está presente nos vídeos incorporados via (iframe). O YouTube usa JavaScript para: Carregar e controlar os vídeos, permitindo autoplay, pausa e reprodução. Garantir interatividade, como ajustar tamanho do player e permitir modo tela cheia. Gerenciar eventos, como exibir botões de compartilhamento e recomendação dentro do vídeo.

Corrida

O código usa três linguagens principais: JavaScript, CSS e HTML

JavaScript (com a biblioteca p5.js) Cria o canvas com createCanvas(). Controla a lógica do jogo: movimentação dos jogadores (xjogador1, etc.), verificação de vencedor, reações ao teclado (keyReleased()), etc. Desenha na tela com text(), rect() e background(). Usa a biblioteca p5.js pra facilitar desenho e interação — bem mais intuitivo do que usar só JavaScript puro!

CSS (style.css) Define o visual da página fora do canvas: Remove margens/padding (margin: 0; padding: 0;) pra o jogo ocupar a tela certinho. Deixa o canvas em bloco com display: block pra não quebrar layout.

HTML É o esqueleto da página: Carrega a biblioteca p5.js e o script do jogo (sketch.js). Define a estrutura da página com e . Pode ser usado pra adicionar outros elementos depois, como botões, placares, ou instruções de jogo.
